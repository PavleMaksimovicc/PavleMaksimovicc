### 🚀 Featured Project

**Digital Signage Management System** *(private repo)*
A full-stack platform for managing and distributing video content across a network of Android TV devices in retail stores.

- **Admin panel** (web app) — upload and manage video content, register/monitor TV devices per store, assign videos to devices with scheduling (start/end time, priority), live dashboard with sync/device status
- **Android TV client** (Kotlin) — authenticates via device token, syncs with the server on a polling interval, downloads and verifies content (SHA-256 checksum), plays scheduled videos fullscreen, works offline using previously downloaded content
- **Backend** — REST API (`/api/v1`), JWT auth for admins + device tokens for TVs, PostgreSQL for metadata, object storage for video files
- Built to handle 1,000+ concurrently active devices and 10,000+ videos in the library

*Currently in active development — architecture, API, and role-based access (Super Admin / Content Admin / Store Manager / Read-only) already spec'd out. Kept private for now due to potential commercial use.*
